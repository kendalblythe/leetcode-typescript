## Getting Started

After you clone the Git repository, you can run the following <a href="https://pnpm.io/">pnpm</a>
commands.

- `pnpm install` - install packages
- `pnpm test` - run tests
- `pnpm lint` - lint check
- `pnpm format` - format code
